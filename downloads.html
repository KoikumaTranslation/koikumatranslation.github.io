Get the patch here: patch2.xp3

This interface patch translates most of the interface of the game along with adding some small improvements like an option to skip the prologue and 'uncensoring' of thumbnails, etc.
It is compatible both with installations that have and those that don't have the official version 1.1 patch, however using the official patch is recommended - you can get it from one of the mirrors listed here.

To use this patch, simply put it in the game's directory.
If you find any errors or have any issues, please let me know.
